# Issue Tracker

**Provider:** GitHub Issues
**Repo:** mikeartee/Ngati-Pakahi-O-Manga-Iti

## Conventions

- Issues are referenced as `#N` (e.g. `#1`, `#42`)
- Sub-issues are linked via GitHub's sub-issue feature
- Parent PRD issues carry the `tracking` label
- Slice issues carry `[AFK]` or `[HITL]` prefix in their title
- Wave and letter are encoded in the title: `[AFK] 1a — description`

## CLI

Use `gh issue` commands for all issue tracker interactions.
